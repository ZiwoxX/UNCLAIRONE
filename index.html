BAHASA & TEKNOLOGI

Website ini dibangun tanpa Website Builder (Tanpa Wix, Tanpa Wordpress).
dengan menggunakan beberapa bahasa pemrograman berikut:

HTML V5

Website ini disusun menggunakan HTML V5 dengan struktur markup modern yang memiliki fungsi jelas, yang membuat setiap bagian halaman web tersusun secara sistematis dan lebih mudah dioptimalkan

CSS V3

Fungsi CSS pada web ini adalah untuk menata tampilan dan gaya visual pada web ini. Menggunakan fitur dari CSS yaitu Glassmorphism, Flexbox, dan Grid Layout, untuk tampilan responsif di HP & Laptop.

JavaScript ES6+

Berperan sebagai “otak” dari website ini. JavaScript yang mengatur proses pencarian data siswa, menampilkan data secara dinamis, mengelola fitur jam digital, dan membuat berbagai tombol dapat berfungsi tanpa harus memuat ulang halaman web.

Firebase

Database Cloud dari Google. Berfungsi untuk Menyimpan foto gallery, komentar, dan jumlah Likes secara Real-time (Langsung update). Tanpa Firebase fitur seperti foto gallery, komentar, dan jumlah likes tidak akan bisa berfungsi

GSAP

Engine animasi yang biasa digunakan para web developer profesional. Membuat efek kartu muncul satu per satu dan transisi halus saat website di-scroll.

FITUR UNGGULAN

Kartu Pelajar Digital

Setiap siswa mempunyai profil yang interaktif lengkap dengan biodata, media sosial, galeri foto pribadi, dan motto.

Fitur Apresiasi

Implementasi nilai budaya saling menghargai. Kamu bisa memberikan Like dan menulis Komentar di profil tiap siswa.

Chat Anonim

Kirim pesan rahasia ke teman sekelas secara anonim. Pesan tersimpan aman di Cloud(Firebase) dan muncul di halaman utama web.

Admin Panel

Halaman khusus admin untuk memngatur atau menghapus chat/komentar yang tidak pantas.

"FILOSOFI: PHINISI & LONTARA"

Web ini menggunakan simbol Phinisi sebagai navigasi, dan background partikel Aksara Lontara sebagai pengingat agar tidak melupakan budaya di tengah canggihnya teknologi jaman sekarang.
